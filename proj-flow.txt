Detailed steps

1. create a github repo and clone it on your local
2. Add experiments folder and add "my_notebook.ipynb"
3. Add src folder and add "data_ingestion.py" (run them individually)
4. Now add "preprocessing.py" inside src
5. Now add "feature_engineering.py" inside src
6. Now add "model_training.py" inside src
7. Now add "model_evaluation.py" inside src
8. Add data, models, reports, directories to .gitignore file
9. Now do git add, commit and push


setting up dvc pipeline (without params)
10. create a dvc.yaml and add the stages to it
11. dvc init then do "dvc repro" (reproduce) to test the pipeline automation (check dvc dag)
12. Do git add and push

setting up dvc pipeline (with params)
13. add params.yaml file
14. Add the params setup in dvc.yaml and the below config to respective files (data_ingestion, feature_engineering, model_training)
15. Do "dvc repro" again now to test the pipeline along with the params
16. Now git add and push

params setup
1. import yaml
2. add func:
def load_params(prams_path: str) -> dict:
    """
    Load parameters from the YAML file
    """
    try:
        with open(params_path, 'r') as file:
            params = yaml.safe_load(file)
        logger.debug("Parameters retrieved from %s", prams_path)
        return params
    except FileNotFoundError:
        logger.error("File not found: %s", prams_path)
        raise
    except yaml.YAMLError as e:
        logger.error("YAML error: %s", e)
        raise
    except Exception as e:
        logger.error("Unexpected error: %s", e)
        raise
3. Add to main()
